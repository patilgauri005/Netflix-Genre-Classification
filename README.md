# Netflix Genre Classification using Machine Learning

## Project Overview
This project implements a Machine Learning model to classify Netflix movies and TV shows based on their genre. The model is built using the Decision Tree Classifier algorithm and trained on the Netflix Movies and TV Shows dataset obtained from Kaggle.

The objective of this project is to analyze Netflix content data and automatically classify or predict genres using supervised machine learning techniques.

---

## Problem Statement
Streaming platforms like Netflix host thousands of movies and TV shows. Organizing and categorizing such large amounts of content manually can be time-consuming and inefficient.

This project aims to:
- Analyze Netflix dataset
- Identify patterns in content features
- Automatically classify genres using Machine Learning

Such systems can help streaming platforms improve content organization and recommendation systems.

---

## Dataset
Dataset used: **Netflix Movies and TV Shows Dataset**

The dataset contains information about Netflix titles such as:

- Title
- Director
- Cast
- Country
- Release Year
- Duration
- Genre (Listed In)
- Type (Movie / TV Show)

Source: Kaggle
